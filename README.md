# My ZSH
Personnal shell configuration for zsh.

Based on kali zshrc.
- auto install with apt
- add .zshrc
- add to default shell with chsh
- user & root
## Quick installation command
Do **not** run as root.
```sh 
bash <(curl -sL https://raw.githubusercontent.com/jojo2massol/myzsh/main/zsh.sh)
```
